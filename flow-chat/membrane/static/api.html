flow / api

accès programmatique à la chasse de patterns

Authentification

Toutes les requêtes nécessitent un header Authorization: Bearer YOUR_API_KEY

curl -H "Authorization: Bearer sk_flow_..." https://pwnd.icu/api/v1/hunt

Endpoints

POST /api/v1/hunt

Lance une chasse de patterns cross-domain

topicstringSujet de recherche
domainsarrayDomaines à croiser ["neuro", "bio", "physics", ...]
depthintProfondeur de recherche (1-3, défaut: 2)
max_papersintNombre max de papers à analyser (défaut: 100)

Exemple

{
  "topic": "bioelectricity morphogenesis",
  "domains": ["neuro", "bio", "physics"],
  "depth": 2
}

Réponse

{
  "patterns": [
    {
      "title": "Voltage gradients encode positional information",
      "confidence": 0.85,
      "sources": ["arxiv:2024.12345", "pubmed:39847562"],
      "connection": ["bio", "physics"],
      "insight": "Gap cells use bioelectric signals..."
    }
  ],
  "papers_analyzed": 87,
  "duration_ms": 12450
}
POST /api/v1/papers

Analyse un batch de papers (upload PDF ou URLs)

papersarrayListe d'URLs ou fichiers base64
focusstringQuestion spécifique (optionnel)
GET /api/v1/search

Recherche dans arxiv/pubmed/scihub

qstringQuery de recherche
sourcestringarxiv, pubmed, scihub, all
nintNombre de résultats (max 50)
POST /api/v1/connect

Trouve les connexions entre deux concepts

concept_astringPremier concept
concept_bstringSecond concept
