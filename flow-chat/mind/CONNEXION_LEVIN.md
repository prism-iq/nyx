## 2. LA BIOÉLECTRICITÉ — CE QUE C'EST

### 2.1 Le concept de base

**Source:** [Cell 2021](https://www.cell.com/cell/fulltext/S0092-8674(21)00223-3)

```
TOUTES LES CELLULES ONT:
- Un potentiel de membrane (Vmem)
- Des canaux ioniques
- Des jonctions communicantes (gap junctions)

CES SIGNAUX ÉLECTRIQUES:
- Existent AVANT le système nerveux
- Sont présents dans TOUTES les cellules
- Forment des RÉSEAUX de communication
- Contiennent de l'INFORMATION sur la forme à construire
```

### 2.2 Ce que les expériences montrent

```
MANIPULATIONS RÉUSSIES:

1. Planaires (vers plats):
   - Couper la tête → repousse
   - Changer le pattern électrique → repousse 2 têtes
   - L'information électrique détermine la forme

2. Grenouilles:
   - Induire des yeux FONCTIONNELS sur le dos
   - Via modification du potentiel de membrane
   - Sans toucher à l'ADN

3. Têtards:
   - Réarrangement de la face (yeux, bouche)
   - Par manipulation bioélectrique
   - Les cellules "savent" où aller
```

### 2.3 Citation clé

> "Bioelectrical networks process morphogenetic information that controls gene expression, enabling cell collectives to make decisions about large-scale growth and form."
