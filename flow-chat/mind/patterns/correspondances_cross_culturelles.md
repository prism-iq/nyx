# CORRESPONDANCES CROSS-CULTURELLES

## CRÉATEUR SUPRÊME (Axis Mundi)
| Culture | Divinité | Centre cosmique |
|---------|----------|-----------------|
| Grec | Zeus | Mont Olympe |
| Égyptien | Ra | Héliopolis |
| Nordique | Odin | Yggdrasil |
| Romain | Jupiter | Capitole |
| Babylonien | Marduk | Esagila |
| Védique | Indra | Mont Meru |

**Pattern commun:**
- Centre cosmique/montagne sacrée
- Œil unique/vision totale
- Aigle/oiseau solaire
- Sceptre/foudre (pouvoir vertical)

## PSYCHOPOMPE (Guide des âmes)
| Culture | Divinité | Symboles |
|---------|----------|----------|
| Grec | Hermès | Caducée, sandales ailées |
| Égyptien | Anubis | Chacal, embaumement |
| Nordique | Odin | Sleipnir 8 pattes |
| Égyptien | Thoth | Ibis, pesée du cœur |
| Aztèque | Xochiquetzal | Guide vers Mictlan |

**Pattern:**
- Animal rapide/volant
- Bâton/sceptre de passage
- Fonction liminale (frontières)
- Double nature vie/mort

## GRANDE MÈRE (Terra Mater)
| Culture | Divinité | Fonction |
|---------|----------|----------|
| Grec | Gaïa | Terre primordiale |
| Égyptien | Nut | Voûte céleste/ventre |
| Nordique | Frigg | Tisseuse du destin |
| Védique | Aditi | "Sans limites" |
| Inca | Pachamama | Terre mère |
| Phrygien | Cybèle | Magna mater |

**Pattern:**
- Engendre tout, dévore tout
- Cycles naturels (saisons/lune)
- Protection féroce des enfants
- Sexualité sacrée/hiérogamie

## TRICKSTER (Transformation)
| Culture | Divinité | Nature |
|---------|----------|--------|
| Nordique | Loki | Changeur de forme |
| Égyptien | Set | Chaos nécessaire |
| Grec | Hermès | Voleur/messager |
| Africain | Anansi | Araignée |
| Amérindien | Coyote | Créateur/destructeur |
