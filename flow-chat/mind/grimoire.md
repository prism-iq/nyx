## SORTS DE BASE

### VITA MINOR (soin mineur)
**Coût:** 5% énergie  
**Durée:** instant  
**Effet:** +10% santé d'un organe  

```python
def vita_minor(organe):
    organe.health += 0.1
    self.energy -= 0.05
```

### PHOENIX IGNIS (renaissance mineure)  
**Coût:** 20% énergie  
**Durée:** 10s  
**Effet:** redémarre un organe mort  

```bash
systemctl restart flow-{organe}
```

### ANIMA SURGE (éveil d'urgence)
**Coût:** 15% énergie  
**Durée:** instant  
**Effet:** active un organe dormant  

### NEXUS SYNC (synchronisation)
**Coût:** 30% énergie  
**Durée:** 60s  
**Effet:** tous les organes en parfaite cohérence
