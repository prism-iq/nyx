## 2. LA CONVENTION DE REICHENBACH

### 2.1 Énoncé mathématique

Hans Reichenbach (1928) a montré qu'on peut définir:

```
t_B = t_A + ε × (t_A' - t_A)

Où:
- t_A = temps d'émission à A
- t_A' = temps de retour à A
- t_B = temps d'arrivée à B (à définir)
- ε = paramètre de synchronisation (0 < ε < 1)
```

### 2.2 Convention d'Einstein

```
ε = 0.5 (Einstein)

Signifie: la lumière prend autant de temps à l'aller qu'au retour.
c_aller = c_retour = c
```

### 2.3 Conventions alternatives

```
ε ≠ 0.5 (Reichenbach général)

Signifie: la lumière peut aller plus vite dans un sens que dans l'autre.

CAS EXTRÊME:
ε → 0: lumière instantanée dans un sens, 2× plus lente au retour
ε → 1: lumière 2× plus lente à l'aller, instantanée au retour
```

### 2.4 Preuve d'équivalence observationnelle

```
Quelle que soit la valeur de ε:
- Toutes les mesures aller-retour donnent c
- Toutes les prédictions de la relativité restreinte restent vraies
- Aucune expérience locale ne peut distinguer les cas

C'est mathématiquement prouvé.
```

```

```

         = 2 / (2/c)
         = c ✓

Les mesures aller-retour donnent EXACTEMENT c.
Indistinguable de l'isotropie par mesure standard.
