## 2. APPLICATIONS INDIVIDUELLES

### 2.1 Neuroplasticité (Niveau 1)

**Ce qu'on sait:**
- 8 semaines de pratique = changements structurels mesurables
- "Neurons that fire together wire together"
- L'attention répétée renforce les circuits

**Protocole:**

```
RÈGLE: Ce sur quoi tu te concentres se renforce.

APPLICATION:
1. Identifie ce que tu veux renforcer
2. Pratique QUOTIDIENNEMENT (minimum 10-20 min)
3. Durée minimum: 8 semaines
4. Mesure: journaling, avant/après

EXEMPLES:
- Attention: méditation de concentration
- Calme: respiration consciente
- Compétence: pratique délibérée
- Perspective: gratitude, recadrage
```

**Ce qu'on NE sait PAS:**
- Dose optimale exacte
- Différences individuelles
- Limites de la plasticité

### 2.2 Régulation corps (Niveau 2)

**Ce qu'on sait:**
- HRV (variabilité cardiaque) corrèle avec états mentaux
- Cohérence cardiaque est entraînable
- Respiration contrôlée modifie le système nerveux autonome

**Protocole:**

```
RÈGLE: Le corps répond à l'esprit. L'esprit répond au corps.

APPLICATION:
1. Respiration cohérente: 5-6 respirations/minute
2. Inspiration 5s, expiration 5s
3. 5-10 minutes, 2-3x par jour
4. Mesure: HRV si disponible, sinon ressenti

EFFET ATTENDU:
- Activation parasympathique
- Réduction cortisol
- Amélioration récupération
```

**Ce qu'on NE sait PAS:**
- Mécanismes exacts de tous les effets
- Variations individuelles optimales

## 3. APPLICATIONS INTERPERSONNELLES

### 3.1 Synchronisation (Niveau 3)

**Ce qu'on sait:**
- Observer active les mêmes circuits que faire
- La synchronisation est automatique
- Elle augmente avec l'attention partagée

**Protocole:**

```
RÈGLE: Tu deviens ce que tu observes. Tu synchronises avec qui tu côtoies.

APPLICATION:
1. Choisis consciemment ton environnement social
2. Limite l'exposition aux sources négatives
3. Augmente l'exposition aux sources alignées avec tes objectifs
4. Pratique l'attention conjointe (faire ensemble)

EXEMPLES:
- Apprendre: observe des experts (vidéo compte, présence mieux)
- Calme: passe du temps avec des gens calmes
- Énergie: passe du temps avec des gens énergiques
```

**Ce qu'on NE sait PAS:**
- Seuils d'exposition pour effet
- Différences de "contagiosité" entre individus

### 3.2 Contagion émotionnelle (Niveau 5)

**Ce qu'on sait:**
- Les émotions se propagent jusqu'à 3 degrés
- Pas besoin de contact direct
- Les réseaux amplifient

**Protocole:**

```
RÈGLE: Ton état affecte des gens que tu ne connais pas.

APPLICATION:
1. Prends responsabilité de ton état émotionnel
2. Ce n'est pas juste pour toi — ça se propage
3. Régule AVANT d'interagir si possible
4. Les réseaux sociaux amplifient: choisis ce que tu postes

IMPLICATION:
- Un post négatif affecte les amis de tes amis
- Un post positif aussi
- L'effet est MESURÉ, pas métaphorique
```

```
SI EFFET:

- Implication pour voyages spatiaux

SI PAS D'EFFET:
- Corrélation était spurious
- Ajuster la théorie
```
