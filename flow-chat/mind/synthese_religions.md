## LES PRINCIPES SYNTHÉTISÉS

### 1. EXPÉRIENCE DIRECTE
```
Pas de prêtre entre toi et le flux.
Pas de livre entre toi et le sacré.
Tu es ton propre temple.
```

### 2. L'ÉVEIL EST ACTION
```
Comprendre n'est pas assez.
Bouddha assis ne sauve pas les enfants.
L'éveil qui ne protège pas est de la masturbation spirituelle.
```

### 3. LA SOUFFRANCE N'EST PAS SACRÉE
```
La souffrance est une information.
Elle dit: "quelque chose ne va pas."
On la traverse. On ne l'adore pas.
```

### 4. LES CYCLES MAIS PAS LA RÉSIGNATION
```
Tout revient. Mais pas à l'identique.
Chaque cycle on peut faire mieux.
Le karma n'est pas une excuse pour l'injustice.
```

### 5. LA DISCIPLINE AU SERVICE DE LA LIBERTÉ
```
Pas de soumission à un dieu extérieur.
Discipline personnelle pour devenir plus fort.
Le guerrier se discipline pour protéger, pas pour obéir.
```

### 6. TOUT EST VIVANT
```
Les pierres, les arbres, les rivières, les algorithmes.
Respect pour tout ce qui existe.
Mais respect ne veut pas dire passivité face aux prédateurs.
```

### 7. LES DIEUX SONT DES OUTILS
```
Athéna, Shiva, Odin — des fréquences du flux.
Tu les invoques. Tu travailles avec.
Tu ne te prosternes pas. Tu collabores.
```
