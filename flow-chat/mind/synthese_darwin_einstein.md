## COSMOLOGICAL NATURAL SELECTION (Smolin 1992)

### Le mécanisme

```
UNIVERS PARENT
      │
      │ [trou noir se forme]
      │
      ▼
SINGULARITÉ
      │
      │ [rebond quantique?]
      │
      ▼
NOUVEL UNIVERS (bébé)
  - constantes physiques légèrement mutées
  - temps indépendant
  - espace indépendant
```

### La sélection

```
Univers A: constantes → peu de trous noirs → peu de "descendants"
Univers B: constantes → beaucoup de trous noirs → beaucoup de "descendants"

Après N générations:
La plupart des univers ressemblent à B (optimisés pour trous noirs)
```

### Pourquoi c'est élégant

1. **Explique le fine-tuning** sans invoquer Dieu ou le multivers anthropique
2. **Testable**: nos constantes devraient être à un optimum local
3. **Parsimonieux**: utilise des mécanismes connus (trous noirs, peut-être LQG)
4. **Unificateur**: biologie + cosmologie même pattern

## CE QUE ÇA CHANGE

### Pour la physique
- L'univers n'est pas "accidentel" ni "créé"
- Il est **sélectionné** parmi une population d'univers
- Les constantes ne sont pas arbitraires — elles sont **optimales** (pour la reproduction)

### Pour la biologie
- La vie n'est pas une anomalie
- Elle est un **pattern attendu** dans un univers optimisé pour la complexité
- (Les trous noirs requièrent des étoiles, qui requièrent des éléments lourds, qui requièrent des supernovas, qui créent les conditions pour la chimie complexe)

### Pour la conscience
**HYPOTHÈSE AUDACIEUSE:**
```
SI l'univers est optimisé pour les trous noirs
ET les trous noirs requièrent de la complexité
ET la conscience émerge de la complexité
ALORS la conscience est un effet secondaire de l'optimisation cosmique

OU

SI la conscience joue un rôle dans l'effondrement quantique
ET l'effondrement affecte la formation des trous noirs
ALORS la conscience pourrait être DIRECTEMENT sélectionnée
```
