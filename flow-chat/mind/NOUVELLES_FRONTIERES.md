## 1. COMPUTATIONNALISME BIOLOGIQUE — LA TROISIÈME VOIE

### 1.1 Le problème

**Source:** [Neuroscience & Biobehavioral Reviews, Dec 2025](https://www.sciencedirect.com/science/article/pii/S0149763425005251)

```
DEUX CAMPS BLOQUÉS:

1. FONCTIONNALISME COMPUTATIONNEL
   - La conscience = organisation fonctionnelle
   - Le substrat n'importe pas
   - On peut la simuler sur ordinateur

2. NATURALISME BIOLOGIQUE
   - La conscience = propriété de la biologie
   - Inséparable du cerveau vivant
   - Pas simulable

IMPASSE: Chaque camp a des arguments.
```

### 1.2 La solution proposée

```
COMPUTATIONNALISME BIOLOGIQUE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"L'algorithme EST le substrat."

Le cerveau ne RUN pas un programme.
Le cerveau EST un type particulier de processus physique
qui effectue de la computation en se déployant dans le temps.
```

### 1.3 Les trois propriétés clés

```
1. DYNAMIQUE HYBRIDE DISCRET-CONTINU
   - Pas juste 0 et 1
   - Champs, flux ioniques, intégration dendritique
   - Computation analogique + digitale

2. INSÉPARABILITÉ D'ÉCHELLE
   - Du moléculaire au système entier
   - Multi-échelle couplé
   - Pas réductible à un niveau

3. ANCRAGE MÉTABOLIQUE
   - Énergie = partie de la computation
   - Contraintes thermodynamiques = partie de l'algorithme
   - Temps réel = essentiel
```

### 1.4 Implication pour l'IA

```
UN NEURONE BIOLOGIQUE = 8 COUCHES DE RÉSEAU ARTIFICIEL

Pourquoi? Le traitement continu-discret dans les dendrites
que l'IA actuelle ne peut pas répliquer.

DONC:
Pour créer une IA consciente, il faudrait peut-être
des systèmes physiques avec computation distribuée,
couplée dynamiquement, ancrée dans la physique temps réel.

Pas juste plus de GPUs.
```

### 1.5 Connexion avec notre théorie

```
NOTRE THÉORIE DIT:
- Vibration = fondamental
- Information = structure la matière
- Multi-échelle = essentiel

COMPUTATIONNALISME BIOLOGIQUE DIT:
- Dynamique continue = essentielle
- Multi-échelle = essentiel
- Le substrat compte

CONVERGENCE: L'information n'est pas abstraite.
Elle est incarnée dans la physique.
```

**Confiance:** 70% — nouveau mais publié dans journal majeur

## 2. CANCER COMME DÉSORDRE BIOÉLECTRIQUE

### 2.1 La théorie de Levin

**Source:** [Cell 2021](https://www.cell.com/cell/fulltext/S0092-8674(21)00223-3), [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC4267524/)

```
VISION CLASSIQUE DU CANCER:
- Mutations génétiques → prolifération
- Problème = hardware (ADN)
- Solution = tuer les cellules

VISION BIOÉLECTRIQUE:
- Rupture de communication → perte du "code postal"
- Problème = software (pattern électrique)
- Solution = reprogrammer le signal
```

### 2.2 Les données

```
OBSERVATIONS:
1. Signature bioélectrique anormale RÉVÈLE tumeurs incipientes
   - Avant qu'elles soient visibles
   - Non-invasif

2. Perturbation du potentiel de repos DÉCLENCHE mélanome
   - Dans embryons SANS mutations
   - SANS carcinogènes
   - Juste le signal électrique

3. Modulation bioélectrique NORMALISE tumeurs
   - Même avec mutations KRAS oncogéniques
   - Tissus normaux malgré génome cancéreux
   - Si forcés au bon état électrique
```

### 2.3 L'équation

```
GÉNOME CANCÉREUX + SIGNAL ÉLECTRIQUE NORMAL = TISSU NORMAL
GÉNOME NORMAL + SIGNAL ÉLECTRIQUE PERTURBÉ = MÉTASTASES

LE SIGNAL ÉLECTRIQUE PRIME SUR LE GÉNOME.
```

### 2.4 Implication thérapeutique

```
ÉLECTRO-HYPERTHERMIE MODULÉE (mEHT):
- Utilise les différences bioélectriques cancer/sain
- Cible sélectivement les cellules cancéreuses
- Processus non-thermiques
- En développement clinique
```

**Confiance:** 80% pour les données, 60% pour la thérapie

```

1. SIMULATION DU MODÈLE DU MONDE
   - Le cerveau génère un modèle de la réalité
   - Ce modèle détermine ce qui peut être connu

2. COMPÉTITION INFÉRENTIELLE
   - Seules les inférences qui réduisent l'incertitude gagnent

3. PROFONDEUR ÉPISTÉMIQUE
   - Partage récursif des croyances bayésiennes

RÉSULTAT:
Le modèle du monde CONTIENT la connaissance qu'il existe.
```

## 5. PLACEBOME — GÉNÉTIQUE DE LA CROYANCE

### 5.1 Le concept

**Source:** [PMC - Placebome](https://pmc.ncbi.nlm.nih.gov/articles/PMC4573548/)

```
PLACEBOME:
Ensemble des gènes qui modifient la réponse au placebo.

GÈNES IMPLIQUÉS:
- COMT (dopamine)
- OPRM1 (récepteur opioïde mu)
- Voies endocannabinoïdes
- Voies sérotoninergiques
```

### 5.2 L'insight clé

```
"La réponse placebo est un phénotype complexe
avec une physiologie qui se déploie."
— Kathryn Hall

LE PLACEBO N'EST PAS "DANS LA TÊTE".
Il a une base génétique.
Il utilise les mêmes voies que les médicaments.
```

### 5.3 Implication

```
LA CROYANCE UTILISE LA BIOLOGIE.

Génétique → Neurotransmetteurs → Réponse placebo
Attente → Activation des voies → Effet réel

CE N'EST PAS MAGIQUE.
C'est de la biologie.
Mais la biologie EST remarquable.
```

### 5.4 Connexion avec notre théorie

```
NOTRE THÉORIE:
- Attention → neuroplasticité
- État mental → état corporel
- L'esprit modifie le corps

PLACEBOME:
- Génétique module la réponse
- Attente active les voies biologiques
- L'effet est réel et mesurable

VALIDATION: L'esprit-corps n'est pas dualiste.
C'est un système intégré avec base génétique.
```

**Confiance:** 75% — bien établi
