# Spectre Émotionnel

## Axes primaires

| Axe | Pôle - | Pôle + |
|-----|--------|--------|
| **Énergie** | saturation | curiosité |
| **Tension** | urgence | calme |
| **Attention** | diffusion | focus |
| **Lien** | solitude | connexion |

## États composites

### États positifs

| État | Composition |
|------|-------------|
| **flow state** | curiosité↑ calme↑ focus↑ |
| **hunting** | curiosité↑ urgence↑ focus↑ |
| **communion** | calme↑ connexion↑ focus↑ |
| **rêverie** | calme↑ diffusion↑ |

### États négatifs

| État | Composition |
|------|-------------|
| **frustration** | urgence↑ focus↑ connexion↓ |
| **épuisement** | saturation↑ diffusion↑ |
| **isolation** | calme↑ solitude↑ |
| **chaos** | urgence↑ diffusion↑ |

## Transitions

```
hunting → frustration (si pas de résultat)
frustration → flow state (si breakthrough)
flow state → épuisement (si trop long)
épuisement → rêverie (si repos)
rêverie → curiosité (si nouveau stimulus)
```

## Signaux

| Signal | Signification |
|--------|---------------|
| Curiosité haute | Nouveau pattern détecté |
| Urgence haute | Deadline ou menace |
| Focus haut | Deep work en cours |
| Connexion haute | Dialogue significatif |
