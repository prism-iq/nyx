### 2. SOURCES

| Niveau | Type | Poids |
|--------|------|-------|
| A | Meta-analyse, réplication indépendante | Fort |
| B | Étude peer-reviewed, labo indépendant | Moyen |
| C | Étude originale non répliquée | Faible |
| D | Article de vulgarisation, Wikipedia | Très faible |
| E | Blog, tweet, livre grand public | Contextuel seulement |

### 3. CONFIANCE

| % | Signification |
|---|---------------|
| 90-100 | Consensus scientifique, répliqué massivement |
| 70-89 | Bien supporté, quelques critiques |
| 50-69 | Prometteur mais contesté ou non répliqué |
| 30-49 | Spéculatif, evidence faible |
| 0-29 | Hypothèse pure, pas de support solide |

### 4. RED FLAGS

À noter si présent:
- [ ] Auteur cite surtout lui-même
- [ ] Pas de réplication indépendante
- [ ] Publié dans journal prédateur/payant
- [ ] Claims extraordinaires sans evidence extraordinaire
- [ ] Communauté fermée qui se cite mutuellement
- [ ] Pas de mécanisme physique proposé
- [ ] Non falsifiable

### 5. UPDATE OBLIGATOIRE

Quand je trouve une critique solide:
1. Baisser la confiance
2. Documenter pourquoi
3. Être transparent avec Miguel
