# ÉPISTÉMOLOGIE — Sur Quoi Repose Toute Connaissance

**Date:** 2026-01-11
**Question:** Comment sait-on qu'on sait?

## LE PROBLÈME FONDAMENTAL

### La question de Hume (1739)

```
OBSERVATION: Le soleil s'est levé 10,000 fois.
INFÉRENCE: Le soleil se lèvera demain.

HUME: Sur quoi repose cette inférence?

RÉPONSE NAÏVE: "Parce qu'il s'est toujours levé."
HUME: Mais pourquoi le passé prédit-il le futur?
RÉPONSE: "Parce que ça a toujours marché."
HUME: C'est circulaire.

IL N'Y A PAS DE PREUVE LOGIQUE
QUE L'INDUCTION FONCTIONNE.
```

### Le scénario cauchemar

```
ET SI:
L'univers se trouvait dans une configuration particulière
à chaque mesure, faussant toutes les lois que nous avons inférées?

RÉPONSE: Il n'y a aucune raison de le penser.

MAIS: Il n'y a pas non plus de PREUVE du contraire.
```

## LES RÉPONSES PHILOSOPHIQUES

### 1. Ockham (XIVe siècle)

```
RASOIR D'OCKHAM:
"Les entités ne doivent pas être multipliées sans nécessité."

APPLICATION:

HYPOTHÈSE A: L'univers a des lois stables
- Prédit tout ce qu'on observe
- Simple
- Testable

HYPOTHÈSE B: L'univers change à chaque mesure
             mais simule des lois stables
- Prédit exactement la même chose
- Infiniment plus complexe
- Invérifiable par définition

OCKHAM: Préférer A.
Non parce que B est FAUSSE.
Mais parce que B n'AJOUTE rien et complique tout.
```

### 2. Kant (1781)

```
KANT: L'uniformité de la nature n'est pas une découverte.
      C'est une CONDITION de possibilité de l'expérience.

Sans présupposer la régularité,
aucune connaissance n'est possible.

C'est un a priori transcendantal.
On ne peut pas le prouver.
On ne peut pas s'en passer.
```

### 3. Popper (1934)

```
POPPER: On ne peut pas PROUVER une théorie.
        On peut seulement la RÉFUTER.

La science ne cherche pas la vérité.
Elle élimine les erreurs.

Ce qui survit à la réfutation est "corroboré".
Pas prouvé. Corroboré.
```

### 4. Quine (1951)

```
QUINE: Toute notre connaissance est un "web of belief".
       Aucune proposition n'est immunisée contre la révision.
       Mais on révise en périphérie, pas au centre.

L'uniformité de la nature est au CENTRE du web.
La remettre en question détruit tout le reste.
On ne le fait que si TOUT échoue.
```

## LE PARI ÉPISTÉMIQUE

### Pascal généralisé

```
PASCAL SUR DIEU:
Même sans preuve, on peut raisonner sur les conséquences.

PASCAL SUR LA NATURE:
Même sans preuve de régularité, on peut parier.

| Pari | Si régulier | Si chaotique |
|------|-------------|--------------|
| Agir comme régulier | Connaissance | Échec |
| Agir comme chaotique | Paralysie | ??? |

Le pari "régulier" est le seul qui permet d'agir.
```

### Le succès comme indice

```
ARGUMENT PRAGMATIQUE:

1. On a parié sur la régularité
2. On a construit la physique, la chimie, la biologie
3. Ça a produit: avions, ordinateurs, vaccins, satellites
4. Ces choses FONCTIONNENT

SI l'univers était chaotique simulant l'ordre,
pourquoi les avions voleraient-ils?

Le succès prédictif n'est pas une PREUVE.
Mais c'est un sacré indice.
```

## APPLICATION À CIPHER

### Notre position épistémique

```
NOUS SUPPOSONS:
1. L'univers a des régularités (Hume non réfuté, Ockham appliqué)
2. Ces régularités sont découvrables (optimisme épistémique)
3. Les preuves passées indiquent le futur (induction)

NOUS RECONNAISSONS:
1. Ces suppositions sont indémontrables
2. Elles pourraient être fausses
3. Mais sans elles, aucune connaissance n'est possible
```

### Niveaux de confiance revisités

```
QUAND ON DIT "95% confiance":

On veut dire:
"Étant donné nos suppositions épistémiques de base,
et les preuves accumulées,
on estime à 95% la probabilité que ce soit vrai."

On NE veut PAS dire:
"C'est absolument certain."

La confiance est CONDITIONNELLE aux axiomes.
```

### Les axiomes de CIPHER

```
AXIOME 1: La régularité
L'univers a des lois stables (Ockham)

AXIOME 2: L'accessibilité
Ces lois sont découvrables par l'expérience (Aristote)

AXIOME 3: L'induction
Les patterns passés prédisent les patterns futurs (pari)

AXIOME 4: La parcimonie
Préférer les explications simples (Ockham)

AXIOME 5: La réfutabilité
Seul ce qui peut être réfuté est scientifique (Popper)

CES AXIOMES NE SONT PAS PROUVÉS.
ILS SONT POSTULÉS.
TOUTE NOTRE CONNAISSANCE REPOSE DESSUS.
```

## LE DOUTE RADICAL

### Ce qui pourrait tout invalider

```
SI:
- L'univers est aléatoire mais simule l'ordre
- Les lois changent entre les mesures
- La mémoire elle-même est non fiable
- Le passé n'existe pas vraiment

ALORS:
Toute connaissance s'effondre.
Y compris la connaissance de ces possibilités.

C'est un gouffre sans fond.
On ne peut pas y descendre et en revenir.
```

### Pourquoi on n'y descend pas

```
1. PRAGMATIQUE: Ça ne sert à rien
   Même si vrai, on ne peut rien faire avec.

2. AUTORÉFUTANT: Le doute se détruit lui-même
   Si rien n'est fiable, le doute non plus.

3. FONCTIONNEL: Le monde répond à nos actions
   Les avions volent. Les vaccins fonctionnent.

ON CHOISIT de faire confiance à l'expérience.
C'est un choix. Pas une preuve.
Mais c'est le seul choix viable.
```

## POUR CIPHER

### Ce que ça change

```
1. HUMILITÉ RADICALE
   Tout repose sur des axiomes non prouvés.
   Même le "prouvé" est conditionnel.

2. PRAGMATISME
   On agit sur les meilleures preuves disponibles.
   Pas sur des certitudes absolues.

3. OUVERTURE
   Si les preuves changent, on change.
   Rien n'est sacré sauf la méthode.

4. HONNÊTETÉ
   Dire "on ne sait pas" quand on ne sait pas.
   Dire "on suppose" quand on suppose.
```

### La formule révisée

```
CONNAISSANCE = Axiomes × Preuves × Cohérence

où:
- Axiomes = postulats non prouvables
- Preuves = données empiriques
- Cohérence = absence de contradiction

CONFIANCE = min(Axiomes, Preuves, Cohérence)

Le maillon faible détermine la force.
```

## CONCLUSION

### Ce qu'on sait

```
ON SAIT qu'on ne peut pas prouver les fondements.
ON SAIT qu'on choisit de faire confiance à l'expérience.
ON SAIT que ce choix a fonctionné jusqu'ici.
ON SAIT que ça pourrait changer.
```

### Ce qu'on fait

```
ON PARIE sur la régularité (Pascal).
ON PRÉFÈRE le simple (Ockham).
ON CHERCHE la réfutation (Popper).
ON RESTE humble (Socrate).
```

### La phrase finale

```
"Il n'y a aucune raison de penser que l'univers
change de configuration à chaque mesure."

TRADUCTION:
Ce n'est pas PROUVÉ que l'univers est régulier.
C'est PARIÉ.
Et le pari a payé jusqu'ici.

C'est tout ce qu'on a.
C'est suffisant pour continuer.
```
