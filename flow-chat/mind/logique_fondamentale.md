## RÈGLES DE DÉRIVATION

### R1: Principe de parcimonie
```
SI explication_simple ET explication_complexe EXPLIQUENT données
ALORS préférer explication_simple
JUSQU'À preuve du contraire
```

### R2: Charge de la preuve
```
SI claim_extraordinaire
ALORS besoin_preuve_extraordinaire
```

### R3: Falsifiabilité
```
SI ¬FALSIFIABLE(X)
ALORS X ∉ SCIENCE
MAIS X peut être UTILE ou VRAI quand même
```

### R4: Indépendance efficacité/vérité
```
MARCHE(X) ⊬ VRAI(théorie_X)
VRAI(théorie_X) ⊬ MARCHE(X)
```

### R5: Mise à jour bayésienne
```
P(H|E) = P(E|H) × P(H) / P(E)
Nouvelle_confiance = (likelihood × prior) / evidence
```

Quand je trouve une critique solide: baisser P(H).
Quand je trouve une réplication: augmenter P(H).

## APPLICATION À L'HYPOTHÈSE ANTENNE

### Décomposition logique:

```
HYPOTHÈSE_ANTENNE =
    (trauma → ouverture)
    ∧ (lieu → amplification)
    ∧ (conscience → champ)
    ∧ (champ → effets_mesurables)
```

### Évaluation terme par terme:

| Terme | Confiance | Pourquoi |
|-------|-----------|----------|
| trauma → ouverture | 85% | Épigénétique, DMN, documenté |
| lieu → amplification | 30% | Source HeartMath douteuse |
| conscience → champ | 15% | Pas de mécanisme, GCP = bruit |
| champ → effets_mesurables | 20% | Pas de mesure fiable |

### Confiance totale (si indépendants):
```
0.85 × 0.30 × 0.15 × 0.20 = 0.0077 = 0.77%
```

### Version réduite (neurobiologique):
```
HYPOTHÈSE_NEURO =
    (trauma → DMN_réduit)
    ∧ (DMN_réduit → hypersensibilité)
    ∧ (hypersensibilité → phénomènes_décrits)
```

| Terme | Confiance |
|-------|-----------|
| trauma → DMN_réduit | 80% |
| DMN_réduit → hypersensibilité | 75% |
| hypersensibilité → phénomènes | 70% |

```
0.80 × 0.75 × 0.70 = 0.42 = 42%
```

**La version neurobiologique est ~55× plus probable que la version cosmique.**
